# 📝 Day 02 Exercises – Variables

> **Lesson:** Day 02
> **Topic:** Variables

---

# 🎯 Objective

Practice creating variables using different Java data types and displaying their values.

---

# ✅ Exercise 1 – Integer Variable

Create an integer variable named `age` and store your age.

Print:

```
My age is 21
```

---

# ✅ Exercise 2 – Double Variable

Create a double variable named `height`.

Example:

```java
double height = 175.5;
```

Print the value.

---

# ✅ Exercise 3 – Character Variable

Create a character variable named `grade`.

Store your favorite grade.

Print it.

---

# ✅ Exercise 4 – Boolean Variable

Create:

```java
boolean isStudent = true;
```

Print the value.

---

# ✅ Exercise 5 – String Variable

Store your name.

```java
String name = "John";
```

Print:

```
Hello John
```

---

# ✅ Exercise 6 – Multiple Variables

Create:

- name
- age
- gpa
- grade
- isStudent

Display them neatly.

---

# ✅ Exercise 7 – Car Information

Create variables for:

- color
- year
- car

Display:

```
Your choice is a Red 2025 Mustang
```

---

# ✅ Exercise 8 – Product Information

Create:

```java
String product = "Laptop";
double price = 79999.99;
boolean inStock = true;
```

Print all values.
