# 📝 Day 07 Exercises – If Statements

## 📖 Overview

These exercises are designed to strengthen your understanding of Java's decision-making statements.

Topics covered:

- if
- else
- else if
- Comparison Operators
- Boolean Variables
- String Validation
- Scanner
- User Input

---

# Exercise 1 — Adult or Child

Ask the user for their age.

Display:

```
Adult
```

if age is 18 or older.

Otherwise display:

```
Child
```

---

# Exercise 2 — Senior Citizen

Ask the user's age.

Display:

- Senior (65+)
- Adult (18–64)
- Child (1–17)
- Baby (0)
- Not Born Yet (negative age)

---

# Exercise 3 — Name Validation

Ask the user for their name.

If it is empty display:

```
You didn't enter your name.
```

Otherwise display

```
Hello <name>
```

---

# Exercise 4 — Student Status

Ask:

```
Are you a student? (true/false)
```

Display the appropriate message.

---

# Exercise 5 — Positive or Negative

Ask the user to enter a number.

Display:

- Positive
- Negative
- Zero
