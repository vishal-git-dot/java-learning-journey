# 📝 Day 01 Exercises – Introduction to Java

> **Lesson:** Day 01  
> **Topic:** Introduction to Java

---

# 🎯 Objective

These exercises reinforce the concepts learned in today's lesson, including:

- Java program structure
- `main()` method
- Console output
- `print()` vs `println()`
- Escape sequences
- Comments

---

# ✅ Exercise 1 – Hello Java

Write a Java program that prints:

```
Hello, Java!
```

### Expected Output

```
Hello, Java!
```

---

# ✅ Exercise 2 – Favorite Things

Print the following:

```
My name is Vishal.
I love programming.
Java is fun!
```

Use **three separate `println()` statements**.

---

# ✅ Exercise 3 – Using print()

Using only `print()`, display:

```
Welcome to Java Programming
```

### Hint

Remember that `print()` does **not** move to the next line.

---

# ✅ Exercise 4 – Escape Sequences

Print the following exactly:

```
Java
Programming
Course
```

Use the `\n` escape sequence.

---

# ✅ Exercise 5 – Tabs

Print:

```
Name    Age
John    20
```

Use the `\t` escape sequence.

---

# ✅ Exercise 6 – Quotes

Print:

```
Java says "Write Once, Run Anywhere"
```

Use escaped quotation marks.

---

# ✅ Exercise 7 – Backslash

Print:

```
C:\Users\Java
```

---

# ✅ Exercise 8 – Single-Line Comment

Create a program that contains:

```java
// This is my first Java program
```

The comment should **not** appear in the output.

---

# ✅ Exercise 9 – Multi-Line Comment

Write a multi-line comment describing yourself.

Example:

```java
/*
Name:
Age:
Favorite Language:
*/
```

---

# ✅ Exercise 10 – Mixed Output

Print:

```
I like Pizza.
It tastes amazing!
Buy me Pizza!
```

Try writing it once using:

- `println()`

Then rewrite it using:

- `print()`
