# 📝 Day 03 Exercises - User Input

## 📖 Objective

Practice accepting user input using the `Scanner` class and become comfortable reading different data types.

---

# ✅ Exercise 1 — Read a Name

### Task

Create a program that asks the user for their full name and prints a greeting.

### Example

```
Enter your name: John Doe

Hello John Doe!
```

---

# ✅ Exercise 2 — Read an Integer

### Task

Ask the user for their age and display it.

### Example

```
Enter your age: 21

You are 21 years old.
```

---

# ✅ Exercise 3 — Read a Double

### Task

Ask the user for their GPA.

### Example

```
Enter GPA: 3.85

Your GPA is 3.85
```

---

# ✅ Exercise 4 — Read a Boolean

### Task

Ask whether the user is a student.

### Example

```
Are you a student (true/false): true

Student: true
```

---

# ✅ Exercise 5 — Student Information

### Task

Accept:

- Full Name
- Age
- GPA
- Student Status

Display all information in a formatted output.

---

# ✅ Exercise 6 — Favorite Things

Ask the user for:

- Favorite Color
- Favorite Food
- Favorite Car

Display the answers.

---

# ✅ Exercise 7 — Rectangle Area Calculator ⭐

Ask the user for:

- Width
- Height

Calculate:

```
Area = Width × Height
```

Example:

```
Width : 5.5
Height: 4.2

Area = 23.1 cm²
```
