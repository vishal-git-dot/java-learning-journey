# 📝 Day 04 Exercises - Mad Libs Game

## 📖 Overview

These exercises reinforce the concepts learned while building the Mad Libs Game.

Topics covered:

- Scanner
- User Input
- Strings
- Variables
- String Concatenation
- Console Output

---

# Exercise 1 — Complete the Story

Complete the following story using your own words.

```
Adjective:
_________________

Noun:
_________________

Verb:
_________________

Adjective:
_________________

Story:

Today I visited a __________ zoo.

Inside the zoo I saw a __________.

The __________ was __________ and __________.

I was __________.
```

# Challenge ⭐

Create a Mad Libs program with at least **10 user inputs**.

Make the story as creative and funny as possible.

# Revision Questions

1. What package contains Scanner?

2. Why do we import Scanner?

3. What method reads an entire line?

4. Difference between next() and nextLine()?

5. Why should Scanner be closed?

6. What is String concatenation?

7. Which operator joins Strings?

8. Why are Strings enclosed in double quotes?

9. Can Scanner read integers?

10. Can Scanner read doubles?

After completing these exercises, you should be comfortable with:
